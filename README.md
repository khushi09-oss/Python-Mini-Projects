## Python Mini Projects

A collection of beginner-friendly command-line Python projects.

## Projects Included

1. `calculator.py`
- Performs arithmetic operations between two numbers.
- Supported operations: `+`, `-`, `*`, `/`, `%`, `**`.
- Includes basic handling for division/modulo by zero.

2. `hangman.py`
- A simple Hangman game.
- Randomly selects a word from a predefined list.
- Gives the user 6 attempts to guess the word letter-by-letter.

3. `bank-account.py`
- Simulates a bank account using a `BankAccount` class.
- Menu options: deposit, withdraw, check balance, transaction history.
- Tracks transactions in memory during runtime.

4. `to_do.py`
- Command-line To-Do List manager.
- Add, view, complete, and delete tasks.
- Saves tasks to `todos.txt` so they persist between runs.

5. `Validate_user.py`
- Validates a user name and email.
- Name must be longer than 2 characters.
- Email must contain `@` and one of the allowed top-level domains.

6. `word_frequency_calculator.py`
- Counts word frequency in input text.
- Converts words to lowercase and removes common punctuation.
- Displays words sorted by frequency in descending order.

## Requirements

- Python 3.8+ (recommended)
- No external libraries are required.

## How to Run

Open a terminal in this folder and run any script with:

```powershell
python <script_name>.py
```

Examples:

```powershell
python calculator.py
python hangman.py
python to_do.py
```

## Notes

- These are interactive terminal programs; most scripts require user input.
- File names are case-sensitive on some systems. On Windows, this is usually not an issue.
